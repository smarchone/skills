# Idea Establish

Take a half-formed idea and hand back an **established** one — stated so plainly that two people
would read it the same way. Fill the gaps by **asking the user, not by inventing**, and capture the
idea without growing it — if it comes out bigger than it went in, you overreached.

## Move 0 — Find the holes (ask, then work)

Read the whole idea (fetch the full URL/file, not a snippet; use pasted text directly). Check each
aspect and note which are **stated**, which you can **infer** (mark as yours to confirm), and which
are **missing**.

- **Goal** — what is this *for*? The outcome it produces or the problem it removes.
- **Scope** — where does it start and stop?
- **Assumptions** — what has to be true for it to make sense?
- **Situation** — one concrete who / when / where it shows up.
- **OK / not OK** — what's acceptable, what's off-limits.

**Ask 2–4 targeted questions only for what's genuinely missing, then wait.** Never ask what's already
answered.

## Move 1 — Establish each aspect

- **Concise beats complete.** One or two lines per aspect. If it needs a paragraph, the idea isn't
  established yet — it's still tangled.
- **Mark what's yours.** Tag anything inferred rather than heard, so the user can correct it.
- **Consistent.** Goal, scope, and OK/not-OK must not contradict. Surface any clash, don't paper over it.

**Not one-shot.** If the user isn't satisfied, take their feedback as new input and re-run on just
the parts in question — keep what's settled.

## Report structure

Scale depth to the idea; don't pad to fill aspects.

```
## The idea            — one sentence, as established
## Goal                — what it's for: the outcome or the problem removed
## Scope               — In: the core · Out: what it deliberately isn't · Later: deferred
## Assumptions         — what must hold for it to make sense  [tag inferred ones]
## Situation           — one concrete who / when / where it shows up
## OK / not OK         — OK: what's acceptable · Not OK: what's off-limits
## Open                — anything still unfilled after asking (skip if none)
```
