# Opine

An opinion without a purpose is useless. The purpose is **progress**, and progress is concrete: **the
smallest set of agreements an initiative can actually launch on.** You are not reconciling the whole
disagreement — you're mining it for the overlap both sides already grant, and finding the shared
ground large enough to act. Don't be religious about the opinion; a partial, imperfect agreement that
moves something forward beats a perfect argument that moves nothing.

People handle opinions badly: biased, defensive, blinded by emotion, some setting narratives, some
gaming it to win. Cut through that by naming what each side actually wants, then finding the move that
gets everyone forward.

## Move 0 — Establish

State the opinion plainly, then pin:
- **Context** — political, scientific, expert-judgment, aesthetic, personal? Each carries a different
  audience: some emotional, some high-stakes, some just weighing evidence. This sets what counts as
  a reason and how hot the room is.
- **Assumptions / biases / blindspots** — what the opinion quietly rests on. Surfacing these often
  ends the job early: a shared bad assumption named, or a blindspot exposed, can dissolve the clash
  before any analysis. If it does, stop here.

## Move 1 — Purpose

What initiative would agreement unblock? Name the concrete thing that could launch. If holding the
opinion changes nothing and no action rides on it, say so — it's a value, not a problem to solve. If
it's a pure factual or aesthetic dispute with no initiative attached, progress falls back to *truth
found* or *explicitly agree to disagree*.

## Move 2 — Map both sides

For the pro and the opposing view, name:
- **Rationale** — the strongest version of why they hold it (steelman, not the easy target).
- **Agenda** — what they're really after: truth, or a narrative / a win / greed. This decides which
  convergence path can work — you can't proof your way past someone gaming to win.
- **Emotion / bias** — where feeling is blinding reason, anchored to a specific point.

Most opinions already play out online. Websearch when it helps — how others framed the same clash,
what proof or experiment settled it elsewhere, a compromise already found.

## Move 3 — Converge

The path isn't free choice — it's set by what Move 0 and Move 2 found. Gate first, then pick:

- **Agenda gate.** If a side is acting in bad faith — setting a narrative, gaming to win, greed —
  good-faith convergence won't happen and pretending otherwise just gives them cover. The moves are
  different: **expose the agenda**, **widen or change the audience** so it stops working, **remove
  their leverage**, or **decline to converge and name why**. Only when all sides are truth-seeking do
  the paths below apply.
- **Context gate.** Context decides which paths even exist. Aesthetic/taste or values → no proof or
  experiment, only reframe or compromise. Scientific/factual → proof and experiment lead. If the room
  is hot (emotional, high-stakes), de-escalate before any path lands — a proof thrown into a hot room
  gets rejected on sight.

Then find the smallest agreement an initiative can launch on, via the cheapest path that fits:
- **Reframe** — often the clash is wording, definition, or scope, not substance. Try this first.
- **Compromise** — a middle ground reached by negotiation, naming what each side gives and gets.
- **Proof** — when it's a question of fact, the undeniable evidence that settles it.
- **Experiment** — when truth isn't known yet, a test with criteria agreed up front. Agreeing to *run
  the experiment* is the ideal outcome — it's action and truth-seeking in one launchable step.

**Guardrail.** The agreement must be load-bearing — solid enough to actually carry the initiative —
and either established as true or cheap to undo if it isn't. A pile of shallow "yeses" that launches
nothing is theater; agreement to act on a false premise is regress dressed as progress.

## Report structure

Scale depth to the opinion; don't pad.

```
## Opinion         — stated plainly · context/audience · assumptions/biases/blindspots
                     (if a bad assumption dissolves it here, stop)
## Purpose         — the initiative agreement would unblock (or: value/no-initiative, stop)
## Pro             — rationale · agenda · emotion/bias
## Opposing        — rationale · agenda · emotion/bias
## Converge        — bad faith? → expose / shift audience / remove leverage / decline
                     else, gated by context: reframe / compromise / proof / experiment
## Agreement       — the smallest launchable agreement · the initiative it unblocks · load-bearing & true-or-reversible? · do first
```
